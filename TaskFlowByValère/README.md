## Présentation

TaskFlowByValere est une application web de gestion de projets et tickets,
inspirée des outils utilisés en entreprise (Jira, Azure DevOps).
Développée dans le cadre de mon portfolio personnel pour démontrer
mes compétences en développement .NET.

## Fonctionnalités

- **Gestion des projets** — Créer, modifier, supprimer des projets
- **Gestion des tickets** — Suivi des tâches avec priorité et statut
- **Authentification** — Inscription et connexion sécurisée
- **Interface responsive** — Design Bootstrap 5

## 🚀 Lancer le projet en local

### Prérequis
- Visual Studio 2022
- SQL Server LocalDB
- .NET 10 SDK

### Installation

1. Cloner le repo
\`\`\`bash
git clone https://github.com/ValereSoisse/TaskFlowByValere.git
\`\`\`

2. Ouvrir `TaskFlowByValere.sln` dans Visual Studio

3. Appliquer les migrations
\`\`\`bash
Update-Database
\`\`\`

4. Lancer avec **F5**
